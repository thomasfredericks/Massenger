INSTALLATION DANS WINDOWS
========================================
Tout installer en 32 bit ou 64 bit
Installer Visual Studio Express : http://www.visualstudio.com/downloads/download-visual-studio-vs#d-2013-express
Installer Python

Installer node.js et npm (qui est inclu avec node)
Installer serialport par npm (à partir d'une invite de commandes)
Installer websocket par npm (à partir d'une invite de commandes)
Rédémarrer l'ordinateur

CONFIGURER ET DÉMARRER LE SERVEUR
=========================================
Brancher l'Arduino à l'ordinateur serveur
Télécharger le code "massenger_digitalWrite_analogRead" (un des exemples de la logithèque Massenger) sur l'Arduino 
Configurer ArduinoASCIIBridge.js pour qu'il pointe vers le bon port de l'Arduino (var ARDUINO_PORT = '???';)
Démarrer node avec le code ArduinoASCIIBridge.js (à partir d'une invite de commandes)

CONFIGURER ET DÉMARRER LE CLIENT
=========================================
Configurer index.html pour qu'il corresponde à l'adresse du serveur (var adresseDuServeur = "localhost:8080";)
Ouvrir index.html dans un navigateur Web de l'ordinateur client
